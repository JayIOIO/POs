# PROJECT REQUIREMENTS DOCUMENT (PRD)

## Project Name

Sari-Sari Store POS System

## Objective

Build a complete Point of Sale (POS) web application for a Filipino sari-sari store using Vanilla HTML, CSS, JavaScript, Node.js, Express, and SQLite. The system must work offline on a local computer and store all data in a local SQLite database.

---

## Technology Stack

### Frontend

* HTML5
* CSS3
* Vanilla JavaScript
* Responsive Mobile-First Design

### Backend

* Node.js
* Express.js

### Database

* SQLite

### Deployment

* Localhost Deployment
* No Internet Required
* Runs on Windows PC

## Deliverables

Generate a complete production-ready project including:

1. Full Frontend (HTML, CSS, JavaScript)
2. Express Backend API
3. SQLite Database Integration
4. Authentication System
5. POS Module
6. Inventory Module
7. Reporting Module
8. Receipt Printing
9. Backup & Restore System
10. Responsive Mobile Design

The final output must be fully functional, organized, and ready to run locally using:

npm install
npm start

No frameworks such as React, Vue, Angular, Bootstrap, or Tailwind. Use only Vanilla HTML, CSS, JavaScript, Express.js, and SQLite.
